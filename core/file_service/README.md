# File Service 
```
Run with gin-gonic framework
```

## How To Run
### 1. Install go
```
https://go.dev/doc/install
```

### 2. Create go module
```
make init
```

### 3. Install library
```
make install-go-tools
```

### 4. Run server
```
make run
```
File Server will run at localhost and listen on port 8080
